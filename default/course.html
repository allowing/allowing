主要学习课程

名称 概述 掌握后的技能
HTML HTML是一种用来写网页的标记语言 能编写HTML文档
CSS CSS是用来写样式的，网页的外观就靠它来装饰 会装饰HTML文档
JavaScript JS是客户端的脚本语言，主要运行在浏览器，用途主要是表单验证、编写炫酷动画效果等 能编写JS脚本，掌握了一定编程思想
jQuery jQuery是一款必学的JS库，举世闻名，主要用来操作DOM 能轻松增删查差DOM节点
PHP PHP是一门用来编写动态web程序的语言，流行易学 能编写动态生成的HTML文档
MySQL MySQL是一种关系型数据库，用途是存储数据，免费实用的一款数据库 能设计数据库结构
yii2 yii2是一款PHP框架，简单、高效、安全 掌握高级PHP开发技能
HTTP协议 HTTP是一款网络通讯协议 知道什么是协议，举一反三
composer composer是PHP类库的依赖管理工具 会设计可复用的包
git git是一个版本控制系统，在它的帮助下写代码不会写散写乱 妈妈再也不用担心我的代码一团糟了
PhpStorm PhpStorm是PHP开发集成环境，主要用来编写PHP程序 学了这个编辑器，其他语言（java，安卓）的编辑器也会用了
Linux系统 Linux系统主要用在服务器领域 爱上了命令行操作
github github是一个网站，全球最大的代码托管平台 代码仓库能全球漫游
markdown markdown书写文档的标记语言 爱书写博客
